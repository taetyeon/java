데이터 입출력 방법

- JS를 통해 HTML이나 콘솔로 데이터를 출력하는 방법

데이터 출력법

1. document.write() 활용

- 브라우저(html 문서)로 문구를 출력하는 가장 기본적인 방법

2. window.alert()

- 브라우저의 메세지창으로 출력하는 방법 (window가 생략 가능함)
- 최근 사용자들의 사용성 측면에서 좋지 않아 활용되지 않는 기법, 학습/디버깅 용도로 활용

3. innerHTML ★★★★★

- html 태그 객체를 찾아 객체 내부에 html로 출력하는 방법
- 주로 id를 찾아서 html 태그를 조작하는 것이 일반적
기본 문장

4. console.log()

- 개발자 도구 화면에 콘솔로 출력하는 방법
- 일반 사용자에게는 보여지지 않지만, 개발이 끝나면 정리하는 것이 바람직!

데이터 입력법

1. HTML 태그에서 데이터를 가져오는 방법 ★★★★★

- html 태그중 input이 가능한 태그에서 사용자 입력값을 읽어 오는 방법
- 주로 Server에 전송하기 전에 사용자 입력값을 체크하는 용도로 활용


2. window.confirm()

- 메세지 창을 통해 true/false으로 확인하는 방법, 사용성 이슈로 잘 쓰이지 않는다.

3. window.prompt()

- 사용자에게 텍스트를 입력받는 방법, 사용성 이슈로 잘 쓰이지 않는다. 2
